# UPI Transactions Analysis Dashboard

## Project Overview

The **UPI Transactions Analysis Dashboard** is an interactive Power BI project developed to analyze digital payment transactions across banks, cities, payment methods, and customer demographics. The dashboard transforms transaction data into meaningful business insights, enabling organizations to monitor transaction trends, customer behavior, and banking performance.

The project demonstrates how Business Intelligence can be used to understand digital payment ecosystems and support data-driven decision-making in the financial technology sector.

---

## Business Problem

With the rapid growth of digital payments, financial institutions need to monitor transaction patterns, customer behavior, and banking performance efficiently.

This project answers business questions such as:

- Which banks process the highest number of UPI transactions?
- Which cities generate the highest transaction volume?
- How do customers prefer to make digital payments?
- Which devices are commonly used for UPI transactions?
- How do transaction trends change throughout the year?
